🚀 Setup Required

To use this chat demo, you need to configure your OpenAI API key:

  1. Get an API key from OpenAI Platform
  2. Add it to src/environments/environment.ts
  3. Restart the development server

⚠️ Important: Direct browser calls to OpenAI API may face CORS restrictions. For production use, consider implementing a backend proxy.
